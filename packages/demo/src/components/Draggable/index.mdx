# Draggable 拖拽

## 基本示例

<CodeBlock line={''} code={demoObj.basic}>
  <DndContext onDragEnd={props.onDragEnd}>
    <DraggableItem className="demo-draggable-item" left={props.coordinates.x} top={props.coordinates.y}>
      draggable
    </DraggableItem>
  </DndContext>
</CodeBlock>

## 设置拖动点

<CodeBlock line={''} code={demoObj.basic}>
  <DndContext onDragEnd={props.onDragEnd}>
    <DraggableItem className="demo-draggable-item" left={props.coordinates.x} top={props.coordinates.y} handle>
      draggable
    </DraggableItem>
  </DndContext>
</CodeBlock>

## 拖放示例

<CodeBlock line={''} code={demoObj.droppable}>
